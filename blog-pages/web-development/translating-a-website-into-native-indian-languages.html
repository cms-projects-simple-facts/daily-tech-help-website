This is the fastest way to enable language translation without writing multiple language versions of your site.

Step 1: Add the following code in your HTML where you want the language selector to appear:

<div id="google_translate_element"></div>

Step 2: Add this script before the closing </body> tag:


<script type="text/javascript">
  function googleTranslateElementInit() {
    new google.translate.TranslateElement({
      pageLanguage: 'en',
      includedLanguages: 'en,hi,ml,ta,te,bn,gu,kn,mr,pa',
      layout: google.translate.TranslateElement.InlineLayout.SIMPLE
    }, 'google_translate_element');
  }
</script>
